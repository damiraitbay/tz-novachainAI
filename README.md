# Next.js Landing (Test Task)

Лендинг на Next.js + Tailwind CSS в визуальном стиле референса Serendale:
- Hero-блок в неоновой стилистике
- Адаптив: desktop / tablet / mobile (320px+)
- Дополнительные секции: преимущества, отзывы, footer
- SEO: `title`, `description`, OpenGraph
- JSON-LD `schema.org` для отзывов

## Запуск локально

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run start
```

## Деплой

- Vercel: импортировать репозиторий и нажать Deploy
- GitHub Pages: рекомендуется через `next export` и static hosting workflow (опционально)
